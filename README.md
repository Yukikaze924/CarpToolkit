# Carp Toolkit

Foobar is a Python library for dealing with word pluralization.

## Installation

Use [git](https://git-scm.com/) to install foobar.

```bash
git clone https://github.com/Yukikaze924/CarpToolkit.git
```

## Usage

```powershell
# Build the application with Dotnet
dotnet build

# Run
dotnet run
```
